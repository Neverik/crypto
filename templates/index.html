Hello here!
What you're seeing is an introduction to the Crypto system.

  • Visit 0.0.0.0:5000/mine/ to mine Cryptos.
  • Visit 0.0.0.0:5000/all/ to see all of the transactions.
  • Visit 0.0.0.0:5000/add/[to]_[amount] to add a new transaction.
  • Visit 0.0.0.0/new/ to generate new keypair.
  • Visit 0.0.0.0:5000/update/ to be up to date with Crypto.
  • Visit 0.0.0.0:5000/cont/[IP address]/ to add a new contact.

The system is in active development. You can customize the source code and the wallet as you wish. Try to run some attacks - maybe my system doesn't work so well! Anyway, I'll be pleased to receive some feedback.
